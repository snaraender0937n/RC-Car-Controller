# Complete Code Summary

This project includes **three main code components**:

## 1. RC Car Code
- Controls motors using L298N
- Receives BLE joystick data
- Implements safety features

## 2. Controller Code
- Reads joystick & buttons
- Sends data via BLE
- Advertises as "BLE Gamepad"

## 3. Linking Code
- Handles BLE connection
- Parses gamepad data
- Exposes clean API to RC car code
